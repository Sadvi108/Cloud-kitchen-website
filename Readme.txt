Project Overview
BiteBurst is a digital platform for ordering meals from our cloud kitchen. It showcases the menu, offers ordering functionalities, and provides information about our kitchen, enhancing the user experience for convenient and fast food ordering.

Features
Responsive design for various screen sizes
Menu display with categorized items
Simple and intuitive UI
Interactive ordering functionality

Usage
The website provides a front-end interface for the cloud kitchen. All static files (HTML, CSS, and JavaScript) are located within the repository.
Modify or add to the HTML, CSS, or JavaScript files as needed to customize functionality or appearance.
